# IdeaForge

A social signal scraper for identifying emerging startup ideas.

## Project Structure

- **Code**: This directory (`~/Projects/ideaforge/`)
- **Planning docs**: `./docs/` (symlink to DevVault)

## Key Documentation

- `docs/Context.md` - Current state, blockers, and progress
- `docs/Feature-Plan.md` - Implementation roadmap
- `docs/Agent-Instructions.md` - Multi-agent coordination
- `docs/Daily/` - Daily work logs (auto-generated by git-to-daily)

## Development

```bash
npm install
npm test
npm run build
```

## Environment

Requires `GIT_TO_DAILY_VAULT` environment variable for automatic daily logs.
