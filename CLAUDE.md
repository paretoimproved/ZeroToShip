# IdeaForge

A social signal scraper for identifying emerging startup ideas.

## Project Structure

- **Code**: `~/Projects/ideaforge/` (this directory)
- **Documentation**: `~/DevVault/01-Projects/ideaforge/`

## Key Documentation

All planning docs live in DevVault:

- `Context.md` - Current state, blockers, and progress
- `Feature-Plan.md` - Implementation roadmap
- `Agent-Instructions.md` - Multi-agent coordination
- `Daily/` - Daily work logs (auto-generated by git-to-daily)

## Development

```bash
npm install
npm test
npm run build
```

## Environment

Requires `GIT_TO_DAILY_VAULT` environment variable for automatic daily logs.
